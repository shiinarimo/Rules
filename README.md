# Games包含部分更新相关域名，游戏更新可能消耗大量流量，请注意

- 自建规则会剔除部分与proxy重复的规则（如AWS）。

- 有相同公司的游戏共用域名的情况，通常不会重复列出，拆规则请注意。

- DMM游戏很麻烦，有些游戏还在用ip，有些游戏甚至根本不属于dmm的域名，比如碧蓝，又比如bilibili的日服。所以...碰到不能玩的再说吧
