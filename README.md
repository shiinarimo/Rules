- Gamesは更新用のドメイン名を含め、プロキシを使用してゲームを更新すると、大量のトラフィックを消費する可能性があります

- 自作ルールにはproxyと重複するのを排除しましたから，単独で使用することはできません

- 同じ会社のゲームは一部分のドメイン名を共有します。そのドメイン名はこのルールに一回だけ出します

- DMMゲームのルールはかなり面倒で...問題が発生したときに対処します
